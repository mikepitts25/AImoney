# AI Income Blueprint

A digital product website selling a guide on earning income with AI tools.

## Structure

- `index.html` — Sales/landing page
- `guide.html` — Full 27-chapter guide (product content)
- `thank-you.html` — Post-purchase confirmation page
- `css/styles.css` — Custom styles, animations, print styles
- `js/main.js` — Navigation, FAQ accordion, scroll effects, copy-to-clipboard

## Tech Stack

- HTML5, Tailwind CSS (CDN), vanilla JavaScript
- No build step, no dependencies
- Designed for GitHub Pages deployment

## Deployment

1. Enable GitHub Pages in repository settings (source: main branch, root)
2. Replace `https://buy.stripe.com/test_placeholder` in `index.html` with your real Stripe payment link
3. Site will be live at `https://<username>.github.io/AImoney/`
